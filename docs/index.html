Cleaner, faster, nicer Django forms

A tiny, well‑tested set of widgets & helpers for Django forms. Batteries included, production ready.

Get started PyPI GitHub
1
  • BetterForm & BetterModelForm with fieldsets and per-row row_attrs for tidy layouts.
  • Rich template filters: label, value_text, selected_values, optional, is_checkbox, is_multiple, is_select, is_radio.
  • ClearableFileField/ClearableImageField + ClearableFileFieldsAdmin, ImageWidget with thumbnails, and AutoResizeTextarea.

Ship forms your users enjoy filling — and you enjoy maintaining.

Install

Stable (PyPI)

pip install django-modern-form-utils

Recommended for production.

Preview (TestPyPI)

pip install -i https://test.pypi.org/simple/ django-modern-form-utils

Try pre‑releases or verify a new build.

Add to Django

INSTALLED_APPS = [
    # ...
    "modern_form_utils",
]

No migrations required for basic widgets.

Quickstart

Use the ImageWidget in a ModelForm

from django import forms
from modern_form_utils.widgets import ImageWidget
from .models import Profile

class ProfileForm(forms.ModelForm):
    class Meta:
        model = Profile
        fields = ["avatar"]
        widgets = {
            "avatar": ImageWidget(),
        }

Renders a clean file input with thumbnail preview (when available).

Plain Form usage

from django import forms
from modern_form_utils.widgets import ImageWidget

class UploadForm(forms.Form):
    image = forms.ImageField(widget=ImageWidget())

Drop‑in replacement for the default file widget.

Features

Fully tested

Continuous Integration across multiple Python and Django versions mirrors real‑world projects.

Core components

BetterForm & BetterModelForm

Define fieldsets and per-row row_attrs (classes, styles) for tidy layouts.

Template filters

label, value_text, selected_values, optional, is_checkbox, is_multiple, is_select, is_radio.

Clearable fields

ClearableFileField / ClearableImageField + ClearableFileFieldsAdmin add a “clear” checkbox for files/images.

ImageWidget

Shows thumbnails for image fields (works with sorl-thumbnail or easy-thumbnails if installed).

AutoResizeTextarea

Auto-expands based on content; small jQuery enhancement with sane defaults.

Usage examples

BetterForm

from django import forms
from modern_form_utils.forms import BetterForm

class MyForm(BetterForm):
    one = forms.CharField()
    two = forms.CharField()
    three = forms.CharField()

    class Meta:
        fieldsets = [
            ("main", {"fields": ["two"], "legend": ""}),
            ("Advanced", {
                "fields": ["three", "one"],
                "description": "Advanced fields",
                "classes": ["advanced", "collapse"],
            }),
        ]
        row_attrs = {
            "one": {"style": "display: none"},
        }

ClearableFileField

from django import forms
from modern_form_utils.fields import ClearableFileField

class MyModelForm(forms.ModelForm):
    resume = ClearableFileField()

ImageWidget

from django import forms
from modern_form_utils.widgets import ImageWidget

class MyForm(forms.ModelForm):
    avatar = forms.ImageField(widget=ImageWidget())

AutoResizeTextarea

from django import forms
from modern_form_utils.widgets import AutoResizeTextarea

class MyForm(forms.Form):
    description = forms.CharField(widget=AutoResizeTextarea())

Template filters

{% load modern_form_utils %}
{{ form|render }}
{{ form.fieldname|label:"Custom Label" }}
{{ form.fieldname|value_text }}
{% if form.fieldname|is_checkbox %}...{% endif %}

Available: label, value_text, selected_values, optional, is_checkbox, is_multiple, is_select, is_radio.

Admin integration

Clearable in admin

from modern_form_utils.admin import ClearableFileFieldsAdmin

class MyAdmin(ClearableFileFieldsAdmin):
    pass

ImageWidget in admin

from django.db import models
from modern_form_utils.widgets import ImageWidget

class MyAdmin(admin.ModelAdmin):
    formfield_overrides = {
        models.ImageField: {"widget": ImageWidget},
    }

Settings

Override the jQuery used by AutoResizeTextarea if needed:

JQUERY_URL = "https://code.jquery.com/jquery-3.6.0.min.js"

If unset, defaults to https://ajax.googleapis.com/ajax/libs/jquery/1.8/jquery.min.js.
